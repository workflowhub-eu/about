description [?] This is the Profile Crate for Workflow RO-Crate. Workflow RO-Crates are a specialization of RO-Crate for packaging an executable workflow with all necessary documentation. It is aligned with, and intends to strictly extend, the more general Bioschemas ComputationalWorkflow profile. WorkflowHub uses Workflow RO-Crates as an exchange format for users to upload a packaged workflow.
